# Projects

AGE Forge is building tools and resources for the Apache AGE ecosystem.

## Active Projects

### AgeORM

A Python ORM for Apache AGE, inspired by [arango-orm](https://github.com/kashifpk/arango-orm). Define your graph models as Python classes and interact with AGE using an intuitive API.

**Status:** Design phase
**Repository:** *Coming soon*

[Learn more →](./age-orm)

---

### Benchmarks

Fair, reproducible benchmarks comparing Apache AGE with other graph databases. All tests run in containers with equivalent configurations.

**Status:** Initial benchmarks complete
**Repository:** *Coming soon*

[Learn more →](./benchmarks)

---

### Web UI Research

Evaluating existing tools (like AGE Viewer) and researching modern alternatives for visualizing and interacting with AGE graphs.

**Status:** Research phase

[Learn more →](./web-ui)

---

## Contributing

All projects are open source. If you're interested in contributing:

1. Check the project repositories on [GitHub](https://github.com/age-forge)
2. Look for issues tagged `good first issue`
3. Join the discussion in GitHub Issues

We welcome contributions of all kinds: code, documentation, testing, and feedback.
